# 智能切竹机应用图标说明

这个文件说明如何创建 bamboo-cut.png 图标文件。

## 图标要求
- 尺寸: 512x512 像素（推荐）
- 格式: PNG，支持透明背景
- 风格: 现代工业设计，简洁明了

## 设计建议
1. 主要元素：竹子（绿色）+ 切割工具/齿轮（金属色）
2. 背景：透明或深色渐变
3. 配色方案：
   - 竹绿色：#4CAF50, #66BB6A
   - 金属色：#607D8B, #78909C
   - 强调色：#FF5722（切割/警告）

## 创建图标的方法

### 方法1：使用 GIMP/Photoshop
1. 创建 512x512 像素的新文档
2. 绘制竹子轮廓（简化的竹节形状）
3. 添加齿轮或切割刀片元素
4. 应用渐变和阴影效果
5. 导出为 PNG 格式

### 方法2：使用在线图标生成器
1. 访问 https://www.canva.com 或 https://www.figma.com
2. 搜索"bamboo"和"cutting"相关图标
3. 组合创建独特设计
4. 下载为 PNG 格式

### 方法3：使用 AI 图片生成
提示词示例：
"Industrial app icon, bamboo cutting machine, green bamboo with metal gear, 
512x512 pixels, transparent background, modern flat design, industrial style"

## 临时解决方案
如果暂时没有图标，可以使用系统默认图标：
```bash
# 复制系统默认应用图标
sudo cp /usr/share/pixmaps/applications-engineering.png /usr/share/pixmaps/bamboo-cut.png
```

或使用 ImageMagick 创建简单图标：
```bash
# 创建简单的文字图标
convert -size 512x512 xc:transparent \
        -font DejaVu-Sans-Bold -pointsize 80 \
        -fill '#4CAF50' -gravity center \
        -annotate +0+0 '竹切' \
        bamboo-cut.png
```

## 安装图标
创建完成后，将图标文件重命名为 bamboo-cut.png 并替换此说明文件。
